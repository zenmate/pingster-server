# ZenMate Pingster Server

> Server for our new awesome url dependency checker tool.

## Usage

Clone this repository and run your own instance of Pingster server:

```bash
npm install
npm start
# or for development
npm run watch
```

## Endpoints

`GET /auth/github`
`GET /auth/github/callback`
`GET /list`
`POST /rescan`
